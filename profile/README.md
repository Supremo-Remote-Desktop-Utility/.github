## What is Supremo?

Supremo is a versatile, high-performance remote desktop application developed by Nanosystems. It provides a gateway between devices, allowing users to remotely access and control PCs, servers, and mobile devices from anywhere in the world. Designed with simplicity and security in mind, it is an ideal tool for IT professionals, help desk teams, and individuals needing seamless remote assistance or workstation access without complex configuration or firewall setup.

The software is highly portable, consisting of a small executable file that does not require installation to run. This makes it a preferred choice for quick, on-demand support sessions. Despite its lightweight nature, Supremo packs a robust feature set, including full keyboard/mouse control, file transfer, and integrated chat. For long-term needs, it can also be configured for unattended access, allowing you to connect to remote machines at any time without requiring a user on the other end.

Safety is a top priority for Supremo; every session is protected by AES 256-bit encryption and supports UAC compatibility, ensuring that your data remains secure during remote operations. With its cross-platform support across Windows, macOS, Linux, Android, and iOS, Supremo provides a unified experience for managing your entire network of devices efficiently.
